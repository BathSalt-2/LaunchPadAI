# 🚀 LaunchKit AI: The $0 Budget Agency

**The World's First $0 Budget AI Marketing & Business Engine.**

LaunchKit AI is a multi-agent autonomous agency designed to help developers and founders transform raw ideas into professional business kits and acquire their first 100 users with **zero ad spend**. It focuses on organic growth, strategic positioning, and investor-ready assets.

![LaunchKit AI Banner](https://picsum.photos/seed/launchkit-vibrant/1200/400?blur=2)

## 🌟 Key Features

### 🤖 Multi-Agent Autonomous Pipeline
Orchestrates six specialized AI agents working in sync to build your business and growth engine:
*   **🎯 Astra (Lead Strategist)**: Defines business logic, market positioning, and 12-month milestones.
*   **🎨 Lyric (Creative Director)**: Designs brand identities, logos, and high-fidelity pitch decks.
*   **💰 Vector (Financial Analyst)**: Builds startup costs, break-even analysis, and P&L projections.
*   **📣 Echo (Growth Hacker)**: Creates launch calendars, SEO strategies, and high-conversion marketing.
*   **🚀 Nova (Launch Commander)**: Orchestrates Product Hunt and Hacker News launch day playbooks.
*   **🤝 Atlas (Outreach Specialist)**: Drafts cold emails, influencer DMs, and community engagement scripts.

### 🎙️ AI Audio Briefs
Get a high-fidelity audio summary of your business strategy, generated by Gemini's native TTS engine. Features a custom PCM-to-WAV player for seamless playback.

### 📄 Intelligent Document Parsing
Upload your README, Tech Spec, or raw notes. The AI parses your content, extracts core logic, and performs **Gap Detection** to identify missing critical information like pricing models or legal risks.

### 💾 Cloud Sync & Persistence
*   **Google Authentication**: Securely sign in to save your projects.
*   **Project Archive**: Save, load, and manage multiple business kits in your personal dashboard.
*   **Firestore Integration**: Real-time persistence for all generated assets.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   Gemini API Key
*   Firebase Project (for persistence features)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/launchkit-ai.git
    cd launchkit-ai
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env` file in the root directory:
    ```env
    GEMINI_API_KEY=your_gemini_api_key
    ```

4.  **Firebase Configuration**:
    Ensure `firebase-applet-config.json` is present in the root directory with your Firebase credentials.

5.  **Start the development server**:
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:3000`.

## 🛡️ Security & Privacy
LaunchKit AI uses server-side environment variables to protect your API keys. Firebase Security Rules ensure that users can only access their own saved projects. All AI processing is handled via the Google Gemini SDK.
