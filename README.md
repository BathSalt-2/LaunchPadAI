# 🚀 LaunchPad $0 Budget Agency

**The World's First $0 Budget AI Marketing Engine.**

LaunchPad is a multi-agent AI agency designed to help developers and founders acquire their first 100 users with **zero ad spend**. It focuses exclusively on organic growth, community engagement, and viral loops to build a sustainable user base without a marketing budget.

## 🌟 Key Features

### 🤖 $0 Budget Multi-Agent Pipeline
Orchestrates four specialized AI agents working in parallel to build your organic growth engine:
*   **Strategist Agent**: Defines zero-budget positioning and community-first USPs.
*   **Creative Agent**: Crafts high-conversion organic copy for social media and cold outreach.
*   **Analyst Agent**: Identifies the best free channels (Reddit, X, Discord) and niche communities.
*   **Growth Agent**: Develops viral loops and manual-but-scalable growth hacks.

### 🛠️ Organic Growth Capabilities
*   **Deep Dive URL Analysis**: Extracts product insights to fuel organic content.
*   **Viral Social Scripting**: Scripts optimized for organic reach on TikTok, Reels, and X.
*   **Cold Outreach Templates**: High-conversion DM and email scripts for manual outreach.
*   **Community Targeting**: Specific subreddits and Discord servers tailored to your niche.
*   **Product Hunt Tactical Guide**: A $0 budget checklist for a successful launch.
*   **Referral Loop Design**: Built-in mechanisms to turn your first 100 users into a growth engine.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   Gemini API Key

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/launchpad-ai-agency.git
    cd launchpad-ai-agency
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env` file in the root directory and add your Gemini API key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

4.  **Start the development server**:
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:3000`.
